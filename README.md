# Estudos de QA (Manual e Automação)

Repositório central de estudos em Quality Assurance. Inclui documentação de testes manuais, análise de bugs via DevTools, exercícios de SQL e automação de testes.

##  Estrutura do Projeto
* **/testes-manuais**: Relatórios técnicos de bugs encontrados em sites reais (Ex: Campinas NFSe).
* **/sql**: Exercícios de consultas e manipulação de dados.
* **/logica**: Práticas de lógica de programação voltadas para automação.

## Tecnologias e Ferramentas
* Chrome DevTools (Console, Network, Elements)
* SQL (Consultas básicas)
* Lógica de Programação
